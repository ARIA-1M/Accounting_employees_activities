# Учёт деятельности сотрудников

> Десктопная система для автоматизации учёта задач сотрудников, делегирования, контроля статусов и формирования статистики.

---

## О проекте

**Учёт деятельности сотрудников (УДС)** — это десктопное многопользовательское приложение для управления задачами внутри отдела. Система предназначена для руководителей и их подчинённых: позволяет ставить задачи, отслеживать статусы, делегировать поручения и формировать статистику по сотрудникам.

Проект разработан на платформе **Avalonia UI** с использованием **C#** и **PostgreSQL**. Учётные данные пользователей хранятся в зашифрованном виде, реализована ролевая модель доступа (сотрудник / руководитель).

---

## Основные функции

### Для сотрудника
- Просмотр своих задач
- Смена статуса задачи (новая → в работе → ожидание → решена)
- Оставление комментариев к задачам
- Делегирование задач другим сотрудникам с указанием причины
- Создание задач для себя (без права делегирования)
- Фильтрация задач по статусам и датам

### Для руководителя
- Создание задач и назначение исполнителей
- Доступ ко всем задачам отдела
- Комментирование любых задач
- Просмотр статистики по задачам сотрудников (суммарно и по каждому)
- Фильтрация статистики по статусам и датам
- Управление сотрудниками (добавление, изменение ролей)

### Дополнительно
- Хеширование паролей (BCrypt)
- Запоминание входа на устройстве (шифрованное локальное хранилище)
- Прикрепление файлов к задачам (до 15 шт.)
- Выгрузка статистики в Excel и PDF
- Интеграция с GLPI API для выгрузки статистики по заявкам клиентов
- Визуализация статистики в виде диаграмм

## Модели данных

**ER-диаграмма** включает следующие сущности:

- **User** — пользователь (логин, хеш пароля, роль)
- **Role** — роль (сотрудник, руководитель, администратор)
- **Employee** — сотрудник (ФИО, дата рождения, начальник, активность)
- **Task** — задача (название, описание, статус, даты)
- **Status** — статус задачи (новая, в работе, ожидание, решена)
- **Executor** — исполнитель задачи (связь Task-Employee)
- **Comment** — комментарий к задаче
- **File** — файл, прикреплённый к задаче
